Recent Projects

Project 1

User Manage System

The User Management System (UMS) is a web-based application designed to facilitate the management of user accounts. It provides functionalities for user registration, login, profile management, and administrative controls. Built using Java for back-end logic, JSP and Servlets for dynamic web pages, MySQL for database management, and HTML, CSS, and JavaScript for front-end design, the UMS offers a comprehensive solution for handling user-related tasks.

Project 2

Student Management System

The Student Management System (SMS) is a web-based application developed to streamline the management of student-related data within educational institutions. It facilitates tasks such as student registration, course enrollment, grade management, and administrative oversight. Built using Java for back-end logic, JSP and Servlets for dynamic web pages, MySQL for database management, and HTML, CSS, and JavaScript for front-end design, the SMS offers a comprehensive solution for handling student information effectively.

Project 3

EBook App

The Ebook App is a web-based application designed to facilitate the purchase and download of digital books by users. It offers a streamlined experience where users can browse, purchase, and download ebooks, while administrators handle book management and order tracking functionalities. Built using Java for back-end logic, JSP and Servlets for dynamic web pages, MySQL for database management, and HTML, CSS, and JavaScript for front-end design, the Ebook App simplifies the process of buying and accessing digital reading material.

Project 4

ATM interface

The ATM Interface is a Java-based application designed to simulate the functionalities of an Automated Teller Machine (ATM). It provides users with a virtual platform to perform banking operations such as withdrawing cash, depositing funds, checking balances, and transferring money. Built using Java programming language, the ATM Interface offers a user-friendly and secure environment for conducting financial transactions.

Project 5

Temperature Convertor

The Temperature Converter is a web-based application designed to convert temperatures between different units, such as Celsius, Fahrenheit, and Kelvin. Built using HTML for structure, CSS for styling, and JavaScript for dynamic behavior, the Temperature Converter provides users with an intuitive and interactive platform to perform temperature conversions.
